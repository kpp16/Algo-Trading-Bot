# Algo-Trading-Bot

## Project Description
An algorithimc trading bot that trades various stocks using the alpaca API.

## Getting Started

1. ``` pip install -r requirements.txt ```
2. Generate API keys from https://alpaca.markets/
3. ```python Breakout.py```
